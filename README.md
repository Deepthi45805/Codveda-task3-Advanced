🌳 Binary Search Tree (BST) in Java

📌 Description

This project implements a Binary Search Tree (BST) in Java.
A BST is a tree data structure where each node has at most two children, and:

- Left child contains values less than the parent
- Right child contains values greater than the parent

This project helps in understanding data structures, recursion, and tree traversal techniques.

---

🚀 Features

- ➕ Insert nodes into the BST
- 🔍 Search for a value
- ❌ Delete a node
- 🔁 Tree traversals:
  - In-order
  - Pre-order
  - Post-order
